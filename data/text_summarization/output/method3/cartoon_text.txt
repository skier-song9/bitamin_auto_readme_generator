<main>GAN 이란?</main>
<sub>GAN의 정의와 훈련 과정</sub> <content>GAN은 위조 지폐범과 경찰의 관계를 통해 설명되며, Generator는 Discriminator를 속이기 위해 진짜 같은 이미지를 생성하게 된다.</content> <page>3</page>
<sub>손실함수</sub> <content>Generator와 Discriminator 간의 경쟁을 통해 Generator가 진짜 같은 이미지를 생성하도록 훈련된다. 학습 목표는 손실 함수를 최소화하는 것이 아님.</content> <page>5</page>
<sub>Training Step</sub> <content>GAN의 학습 과정에서 메러를 최소화하도록 학습된다.</content> <page>6</page>

<main>CartoonGAN</main>
<sub>Architecture</sub> <content>상세한 아키텍처 설명.</content> <page>8</page>
<sub>Loss function</sub> <content>CartoonGAN에서 사용되는 손실 함수는 실제 만화 이미지와 생성된 이미지 간의 차이를 최소화하는 것을 목표로 한다. 세부 항목으로는 실제 만화 이미지, 제거된 만화 이미지, 생성된 이미지에 대한 판별기 학습 역할이 포함된다.</content> <page>9</page>

<main>Dataset</main>
<sub>Dataset 구성</sub> <content>'원피스' 시리즈 영화에서 약 11시간 30분 분량의 데이터를 추출하여, 9728장의 만화 이미지와 10000장의 실제 이미지를 포함한 데이터셋을 생성. Edge Smoothing 기법을 사용하여 이미지 품질을 향상시킴.</content> <page>11</page>
<sub>Training Step</sub> <content>상세한 학습 단계 설명.</content> <page>12</page>

<main>CartoonGAN 변형</main>
<sub>변형 CartoonGAN ResNet Block</sub> <content>Generator에 사용되는 ResNet Block에서 batch normalization과 residual connection 이후에 ReLU가 적용되는 구조로 변경.</content> <page>14</page>
<sub>변형 CartoonGAN Generator Upsampling</sub> <content>Generator에서 blurring과 batch normalization을 사용하여 upsampling 과정에서 발생할 수 있는 문제를 해결함.</content> <page>15</page>
<sub>Content Loss</sub> <content>Content Loss를 VGG19에서 VGG16으로 변경하고, VGG16의 24번째 layer까지만 사용하여 feature extractor로 활용.</content> <page>16</page>

<main>Experiment</main>
<sub>Pretrain Generator</sub> <content>Real image의 content를 유지하기 위해 Generator를 사전 학습함. epochs 10, batch size 16, Optimizer AdamW 사용.</content> <page>18</page>
<sub>Discriminator & Generator train</sub> <content>Discriminator와 Generator를 함께 학습시킴. epochs 10, batch size 16, Optimizer AdamW 사용. 손실 함수로는 Adversarial Loss, BCEWithLogitsLoss, Content Loss 등이 사용됨.</content> <page>19</page>
