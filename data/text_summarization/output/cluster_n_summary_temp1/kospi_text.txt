<main>비타민 학기 프로젝트 최종 발표 목차</main>
<subtitle>피쳐 중요도 분석을 통한 KOSPI 지수 예측</subtitle>
<content>KOSPI 지수 예측을 위해 XGBLSTM과 GRU 모델을 사용하였고, 하이퍼 파라미터 튜닝 결과 각각의 모델에서 향상된 정확도를 보였음. 변화율 데이터를 활용한 모델이 약 0.05의 성능 개선을 기록하였다.</content>

<subtitle>데이터 고개</subtitle>
<content>발표에서는 종가 데이터를 포함하여 WTI 원유 가격, 다우존스 산업평균지수, 달러 환율 등 다양한 외부 변수들이 KOSPI 지수에 미치는 영향을 분석함.</content>

<subtitle>모델 고개</subtitle>
<content>XGBoost와 GRU 모델을 통해 KOSPI 지수 예측을 위한 다양한 모델 구조를 구축하고, 각 모델의 하이퍼 파라미터 튜닝을 통해 최적의 성능을 도출하였다.</content>

<subtitle>피쳐 간 상호작용 확인</subtitle>
<content>SHAP을 이용하여 피쳐 간의 상호작용을 확인하였으며, 그 결과 피쳐 간의 상호작용 정도는 미미하다는 분석이 이루어졌다.</content>

<subtitle>결론</subtitle>
<content>피쳐 중요도 분석과 여러 모델의 성능 비교를 통해 KOSPI 지수 예측의 정확성을 높이기 위한 방법을 제시하였으며, Attention 기법을 활용한 피쳐 중요도 분석도 중요한 기여를 하였다.</content>
<main> LEVEL 2 모델 소개 XGB LSTM GRU</main>
<subtitle>LSTM 모델 구축</subtitle>
<content>두 개의 LSTM 레이어를 사용하고 각 레이어 후에 드롭아웃을 추가하여 과적합을 방지하며, EarlyStopping을 통해 가장 좋은 가중치를 복원하는 방식으로 모델을 구축하였다.</content>

<subtitle>GRU 모델 구축</subtitle>
<content>GRU 모델 역시 두 개의 GRU 레이어를 사용하며 과적합 방지를 위해 드롭아웃을 추가하고, EarlyStopping을 통해 최적의 가중치를 복원하였다.</content>

<subtitle>XGBoost 성능 비교</subtitle>
<content>결과적으로 XGBoost 모델이 가장 우수한 성능을 보였으며, 변화율 데이터를 사용할 때 성능이 향상되었음을 확인하였고, 이는 변화율 데이터가 더 유의미한 패턴을 제공하기 때문임을 강조하였다.</content>
<main> EVEL 3 피쳐 중요도 분석 Attention SHAP</main>
<subtitle>데이터 전처리</subtitle>
<content>데이터프레임을 병합하고 날짜를 조정하며 결측치를 처리하였다. 슬라이딩 윈도우 기법을 이용해 시간적 순서를 유지하며 과거 데이터를 바탕으로 예측할 수 있는 방식으로 데이터의 패턴을 반영하였다.</content>

<subtitle>Attention</subtitle>
<content>모델을 정의할 때 Transformer 블록을 포함하여 Attention 메커니즘을 활용하고, 가중치를 추출하여 각 피쳐의 중요성을 분석하는 과정을 설명하였다.</content>

<subtitle>SHAP 변수 중요도 분석 결과</subtitle>
<content>SHAP 분석을 통해 중요 피쳐만을 사용하여 모델링한 결과, 기존 모델과 유사한 성능을 유지할 수 있음을 확인하였다. 특히, 상위 20개 피쳐만으로도 성능이 크게 차이 나지 않았음을 보여줬다.</content>

<subtitle>결과 비교</subtitle>
<content>Xgboost와 LSTM 모델의 성능 비교에서, 변수 중요도 계수를 적용한 경우 Xgboost가 우수한 성능을 보였으나, LSTM의 경우 중요도가 양수인 피쳐만 사용할 때 더 좋은 성능을 나타내었다. 전반적으로 변수 중요도를 적용한 모델이 성능이 가장 뛰어난 결과를 얻었다.</content>
<main> LEVEL 4 결과 비교 및 결론</main>
<subtitle>[결과 비교]</subtitle>
<content>XGBoost가 가장 우수한 성능을 보여주었으며, LSTM과 GRU는 변수 중요도를 적용한 데이터셋에서 약간의 성능 향상을 보였다. 반면 XGBoost는 어텐션 결과 적용 시 성능이 약간 하락했음을 확인하였다.</content>

<subtitle>[결론]</subtitle>
<content>변화율 데이터가 종가 데이터보다 더 정확한 예측 성능을 보이며, RNN 계열 모델(LSTM, GRU)은 어텐션 메커니즘을 활용해 정확도를 높일 수 있었다. 변수 중요도 분석을 통해 필수적인 피쳐 선택이 가능해지며, 이는 모델 성능 개선에 기여할 수 있다.</content>
